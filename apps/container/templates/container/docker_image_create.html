镜像配置说明

镜像配置用于定义CTF题目使用的容器镜像及其相关参数：

  • 镜像名称：完整的镜像名称，如 nginx, ctf/web-challenge
  • 镜像标签：版本标签，默认为 latest
  • 镜像仓库：镜像来源，默认为 docker.io (Docker Hub)
  • Flag注入方法：选择如何将动态Flag注入到容器中
  • 暴露端口：容器需要映射到外部的端口，多个端口用逗号分隔
